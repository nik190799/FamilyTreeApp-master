EXTRA_WRITE_DATA

const val EXTRA_WRITE_DATA: String

Intent extra key for a boolean indicating whether database writes should be made in this activity. This is useful if we want to make database writes in the calling activity.

If this is not specified when starting the activity, the value of DEFAULT_WRITE_DATA will be used.
