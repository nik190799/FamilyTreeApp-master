invokeDialogAction

private fun invokeDialogAction(which: Int): Unit

Invokes a particular action depending on the dialog option selected.

Parameters

which - the index of the dialog option selected. Indices 0 to 3 inclusive correspond to the options always present, see STANDARD_OPTIONS_TEXT_RES. Indices of at least 4 correspond to the "switch to spouse's tree" options (these are not present if the person has no spouses).
