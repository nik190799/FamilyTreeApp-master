CreatePersonActivity()

This activity provides the UI for adding a new person from the database, with a guided format.

To edit an existing person, EditPersonActivity should be used, not this.

When the user adds all the information and confirms, the data for the new person will be written to the database, and the newly created Person will be sent back to the activity from which this was started as a result.
